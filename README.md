# The Oldest View
This is a promo webpage for indie short film **The Oldest View**, known also as *Project Rolling Giant* - a liminal psychological horror film created by Kendrew Lee (based on the webseries by Kane Parsons) that explores memory and nostalgia.

Hosted at [projectrollinggiant.movieslay.com](https://projectrollinggiant.movieslay.com/).

## Development
Contributing to the repo is easy! Just PR your changes and I'll review them. You can dev the project like any other NextJS app:
```bash
# packages
npm install

# development
npm run dev

# production
npm start
```
